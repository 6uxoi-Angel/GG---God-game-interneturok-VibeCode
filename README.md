# God Game

**God Game** — браузерное расширение для удобной работы с Jitsi и InternetUrok.

Расширение объединяет несколько инструментов в одном popup-окне: быструю смену имени в Jitsi, копирование вопросов из тестов InternetUrok, расчёт предполагаемых итоговых оценок и переключение светлой/тёмной темы.

## Возможности
## Установка https://youtu.be/g5v9Vjg-Rgw

### 1. Имя в Jitsi

Расширение помогает быстро менять отображаемое имя на странице Jitsi.

Возможности:

- просмотр текущего имени;
- ввод нового имени;
- применение имени на активной вкладке;
- сохранение истории использованных имён;
- повторный выбор имени из истории;
- очистка истории;
- обновление связи с активной вкладкой.

### 2. Копирование вопросов InternetUrok

Вкладка копирования вопросов предназначена для страниц с тестами и домашними заданиями InternetUrok.

Расширение умеет находить и копировать:

- текст вопроса;
- условия задания;
- варианты ответа;
- задания с одним вариантом ответа;
- задания с несколькими вариантами ответа;
- текстовые поля ответа;
- задания на соответствие;
- изображения в вопросах;
- логи работы сканера.

Доступные действия:

- **Сканировать** — найти вопросы на странице;
- **Скопировать сейчас** — скопировать текущий найденный вопрос;
- **Включить автокопирование** — автоматически копировать вопрос при переходе;
- **Скачать логи сканера .txt** — сохранить технические логи в файл.

### 3. Расчёт итоговой оценки

Вкладка итоговых оценок работает со страницей электронного журнала InternetUrok.

Расширение собирает оценки по каждому предмету и рассчитывает предполагаемую итоговую оценку по среднему баллу.

Пример округления:

- `4.5` и выше → `5`;
- `3.5` и выше → `4`;
- `2.5` и выше → `3`;
- ниже `2.5` → `2`.

### 4. Светлая и тёмная тема

В правом верхнем углу popup-окна есть кнопка настроек **⚙**.

Через неё можно выбрать:

- светлую тему;
- тёмную тему.

Выбранная тема сохраняется в памяти расширения и не сбрасывается после закрытия popup.

### 5. Иконка расширения

В расширении используются иконки:

- `1.png` — 16×16;
- `2.png` — 48×48;
- `3.png` — 128×128;
- `4.png` — отдельный аватар 64×64 для шапки popup.

## Установка

1. Скачайте архив расширения.
2. Распакуйте архив в отдельную папку.
3. Откройте в браузере страницу:

   ```text
   chrome://extensions/
   ```

4. Включите **Режим разработчика**.
5. Нажмите **Загрузить распакованное расширение**.
6. Выберите папку с файлами расширения.
7. Расширение появится в списке установленных расширений.

## Как пользоваться

### Смена имени в Jitsi

1. Откройте страницу Jitsi.
2. Нажмите на иконку расширения.
3. Перейдите во вкладку **Имя в Jitsi**.
4. Введите новое имя.
5. Нажмите **Применить имя**.

### Копирование вопросов

1. Откройте страницу домашнего задания или теста InternetUrok.
2. Нажмите на иконку расширения.
3. Перейдите во вкладку **Копирование**.
4. Нажмите **Сканировать**.
5. После нахождения вопросов нажмите **Скопировать сейчас** или включите автокопирование.

### Расчёт итоговых оценок

1. Откройте страницу журнала InternetUrok с таблицей оценок.
2. Нажмите на иконку расширения.
3. Перейдите во вкладку **Итоговые**.
4. Нажмите **Посчитать оценки**.
5. Расширение покажет список предметов, средний балл и предполагаемую итоговую оценку.

## Структура файлов

```text
God Game/
├── manifest.json
├── popup.html
├── popup.js
├── content.js
├── injected.js
├── 1.png
├── 2.png
├── 3.png
└── 4.png
```

### Описание файлов

- `manifest.json` — описание расширения, разрешения, иконки и подключаемые скрипты.
- `popup.html` — интерфейс popup-окна.
- `popup.js` — логика popup-окна, вкладок, тем, кнопок и обмена сообщениями.
- `content.js` — скрипт, который работает на страницах сайтов.
- `injected.js` — дополнительный скрипт для взаимодействия со страницей.
- `1.png`, `2.png`, `3.png` — иконки расширения.
- `4.png` — аватар 64×64 в шапке popup.

## Разрешения

Расширение использует следующие разрешения:

```json
[
  "activeTab",
  "scripting",
  "storage",
  "clipboardWrite",
  "microphone"
]
```

Для чего они нужны:

- `activeTab` — работа с активной вкладкой;
- `scripting` — внедрение content script при необходимости;
- `storage` — сохранение истории, темы и настроек;
- `clipboardWrite` — копирование вопросов в буфер обмена;
- `microphone` — функция проверки/воспроизведения звука для микрофона.

## Поддерживаемые сайты

Основные сценарии рассчитаны на:

- Jitsi;
- InternetUrok.

Некоторые функции могут работать только на страницах с подходящей HTML-структурой.

## Примечание

Расширение не является официальным продуктом Jitsi или InternetUrok. Оно предназначено для личного удобства и автоматизации действий в браузере.
